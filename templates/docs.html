OpenRouter — Free models

Access dozens of models from one API key, including free tiers

Get Key
  1. Go to openrouter.ai/keys and sign up (free)
  2. Click Create Key — copy the key starting with sk-or-
  3. In NL2SQL → Settings → select OpenRouter, paste the key, click Save
  4. Add a model with one of the slugs below

Free API key from Google AI Studio, generous rate limits

Get Key

  2. Sign in with your Google account
  3. Click Create API Key — copy it
  4. In NL2SQL → Settings → select Google Gemini, paste the key, click Save

Tips for better results

  • → When adding a database, fill in the Schema Description — it helps the AI write more accurate queries.
  • → Be specific: "Show me monthly revenue for Q1 2024" works better than "show revenue".
  • → Use the Dashboard button to generate charts — ask things like "bar chart of top 10 products by sales".
  • → CSV/Excel files are automatically converted to SQLite — each sheet becomes a table.
  • → Toggle Show SQL to see the exact query that was run.
