What you unlock

💬

Natural language chat

Ask anything — "top 10 customers by revenue", "average order value last month". Get a table, a summary, and the SQL it ran.

📊

Power BI–style dashboards

Describe a chart and the AI builds it — bar, line, pie, scatter, radar. Pin charts to a persistent dashboard per database. Expand any chart fullscreen for presentations.

📁

Upload & query instantly

No database server needed. Drop a CSV or Excel file — it becomes a full SQLite database you can query with plain English straight away.

🔁

Self-correcting AI

If a query fails, the AI checks the schema, corrects the SQL, and retries automatically — up to 8 times. You almost never see an error.
