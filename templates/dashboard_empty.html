Talk to your databases.
No SQL required.

Hey — welcome to NL2SQL. I built this because I was tired of writing the same SELECT queries every day just to answer simple business questions. Now you just… ask. In plain English. The AI figures out the SQL, runs it on your database, and hands you back the answer with a table and a chart.

It works with PostgreSQL, MySQL, SQL Server, SQLite — and yes, you can also just drag-and-drop a CSV or Excel file and start querying it immediately. No database server required for that.

Two steps to your first answer

1

Add an AI model

NL2SQL needs an LLM to turn your question into SQL. The free options below work great. Go to Settings, pick a provider, paste the key, and add a model. Takes under a minute.

Open Settings
2

Connect a database (or upload a file)

Connect PostgreSQL, MySQL, SQL Server, or a local SQLite file. Or skip the database entirely — upload a CSV or Excel file and it becomes a queryable database instantly. Great for testing.

What you can do once you're set up

This is the part that's actually fun.

💬

Chat with your data

"Show me the top 10 customers by revenue" → instant answer with table and summary.

📊

Build dashboards

Ask for a chart — the AI picks the right type and builds it with ECharts. Save it to your dashboard.

📁

Upload CSV / Excel

No database server needed. Drop a spreadsheet and query it like a database.

🔁

Self-correcting AI

If the first query fails, the AI checks the schema, fixes the SQL, and tries again — up to 8 times.
